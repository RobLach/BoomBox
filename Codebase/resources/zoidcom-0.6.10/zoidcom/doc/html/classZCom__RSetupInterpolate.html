Member Data Documentation

template<typename T>
zFloat ZCom_RSetupInterpolate< T >::ipol_factor
 

Interpolation factor.

A value of 1.0 always overwrites the existing data with the incoming update. A value of 0.0 ignores incoming updates and leaves the local value as is.

template<typename T>
T ZCom_RSetupInterpolate< T >::ipol_treshold
 

Interpolation treshold.

If difference between incoming data and local data is greater than the treshold, no interpolation occurs. Instead, the local value will be replaced with the incoming value.
