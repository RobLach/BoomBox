Purpose of this manual

The purpose of this manual is to provide a tutorial-like introduction to all major features of Zoidcom and providing a big picture of how everything works together. It is not meant to be a complete reference to all and every single function in existence, the API docs are better suited to fulfill this task.

The examples will mostly build up on each other, so reading from front to back is recommended.
